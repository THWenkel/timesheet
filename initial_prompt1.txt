# what is the web app being built for?
the user will start in the frontend website and see a react-calender (npm package), a textbox to enter a value for time (hh:mm in 15 min steps) and to enter a free text on what the user has done on a specific date, selected by the react-calender and a save button. Over all on that, there is a field to select a user from a employees table in database. Employees are stored in a table named employees. Surname and Lastname are displayed in that list, but handled separate in the DB.
In the database, a date time should be written to db for created, edited and wich user it was. 
If the user is clicking on a specific date, the data is loaded and displayed in the textboxes (time and free text)
The react-calender should show days in dark blue, if entries for that date exists in the database.
